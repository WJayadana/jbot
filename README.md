# Yumeniru-Md

Base Bot WhatsApp (Case)
by choco


## Run Locally

Clone the project

```bash
  git clone https://github.com/Rasya25/yumenirubot
```

Go to the project directory

```bash
  cd yumenirubot
```

Install dependencies

```bash
  npm install
```

Install dependencies **recommended**
```bash
 npm install --force
```

Start the server

```bash
  npm start
```
